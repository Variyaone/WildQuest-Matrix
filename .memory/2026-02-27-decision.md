## 触发事件

研究员subagent无法读取本地文件，导致无法完成P0任务（知乎策略研究+多因子策略设计）。指挥官代劳完成了任务。

老大指正：为什么不给他授权？

## 根本原因

研究员工具权限配置：
```json
"tools": {
  "profile": "full",
  "allow": [
    "web_search",
    "web_fetch",
    "memory_search",
    "memory_get"
  ]
}
```

缺少`read`、`write`等本地文件操作工具。

## 学到的原则

### 权限管理
- 预先授权优于事后补救
- 根据职责配置工具（最小可用权限原则）
- 高风险功能（外部actions）单独决策

### 执行逻辑
- T0级别：致命/经济损失 → 需要决策
- 非T0级别：直接执行→汇报

### 系统设计
- 创新者常驻：监控系统健康，预判权限需求
- 看板更新：记录配置变更
