## 交易频率的影响因素

### 1. 市场波动率
```
高波动（ETH 85.2%）→ 交易更频繁（56次/90天）
低波动（BTC 72.1%）→ 交易较少（46次/90天）

原因：波动大 → 价格穿越网格线次数多 → 触发交易
```

### 2. 价格走势
```
趋势市场（牛市/熊市） → 交易较少（持续单边）
震荡市场 → 交易较多（上下波动）

示例：
牛市: 492次/6个月 = ~82次/月
震荡: 320次/6个月 = ~53次/月
熊市: 49次/6个月 = ~8次/月
```

### 3. 趋势过滤器
```
熊市触发空仓 → 交易暂停
震荡/牛市 → 网格正常运行

示例：
熊市环境，空仓期间没有网格交易
```
