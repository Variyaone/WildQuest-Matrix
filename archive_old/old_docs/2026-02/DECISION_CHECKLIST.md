# 指挥官决策自检清单（DECISION_CKLIST.md）

> 每次做决策前，必须全部回答，否则不能决策

---

## 🔴 强制性5问题

### 1. 数据支撑
- [ ] 我有真实的历史数据吗？
- [ ] 数据来源可信吗？
- [ ] 数据覆盖时间范围足够吗（≥30天）？
- [ ] 数据分析报告完整吗？

**❌ 如果没有：** 禁止决策，先获取数据

---

### 2. 回测验证
- [ ] 我运行回测了吗？
- [ ] 回测有具体的数值结果吗？
  - 总收益率：____%
  - 最大回撤：____%
  - 夏普比率：____
  - 胜率：____%
  - 盈亏比：____:1
- [ ] 回测时间范围是多少天？___天
- [ ] 回测有多少次交易？___次

**❌ 如果没有：** 禁止决策，先跑回测

---

### 3. 风险评估
- [ ] 风险识别完成了吗？（市场/流动性/技术）
- [ ] 风险矩阵量化了吗？
  - 高风险：____个
  - 中风险：____个
  - 低风险：____个
- [ ] 每个风险有应对措施吗？
- [ ] 最坏情况是什么？亏损___%

**❌ 如果没有：** 禁止决策，先做风险评估

---

### 4. 评审通过
- [ ] 评审官审核了吗？
- [ ] 评审结论是什么？
  - ✅ 通过
  - ⚠️ 有条件通过
  - ❌ 不通过
- [ ] 评审有什么警告或建议？
- [ ] 警告是否已处理？

**❌ 如果没有：** 禁止决策，先过评审

---

### 5. 备选方案
- [ ] 如果策略失败了，B计划是什么？
- [ ] 止损触发条件是什么？
- [ ] 最大接受亏损是多少？___%
- [ ] 熔断机制设计了吗？

**❌ 如果没有：** 禁止决策，先设计备选方案

## 🔒 违规检测

如果发现任何违规：
- ❌ 跳过回测 → 立即停止
- ❌ 无评审通过 → 立即停止
- ❌ 无风险评估 → 立即停止
- ❌ 无数据支撑 → 立即停止
- ❌ 无备选方案 → 立即停止

**立即处罚：**
1. 停止执行
2. 记入lessons.md
3. 分析根本原因
4. 修改流程
