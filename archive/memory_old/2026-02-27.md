## 专业量化机构方法论对标

### 机构化研究流程

```
[数据层] → [因子层] → [策略层] → [组合层] → [风控层] → [实盘层]
    ↓         ↓         ↓         ↓         ↓         ↓
  清洗     挖掘/检验   设计/回测   优化    评估风控   监控调优
```

### 关键模块

| 模块 | 专量化做法 | 简易做法(❌) |
|------|-----------|--------------|
| **数据** | 多源交叉验证、清洗、对齐 | 单源直接用 |
| **因子** |挖掘+IC/IR/正交化 | 简单技术指标 |
| **组合** | 多因子打分、机器学习 | 单指标策略 |
| **回测** | 分年分月、样本内/外 | 简单全样本 |
| **风控** | 行业中性、风格中性、波动率控制 | 止损止盈 |
| **评估** | 夏普、卡玛、信息比率 | 收益率 |
