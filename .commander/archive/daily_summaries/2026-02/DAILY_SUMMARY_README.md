# 📅 每日大事记录系统

**目的**: 记录每天完成的大事，方便老大快速了解进展

---

## 📝 记录规范

### 文件命名

格式：`DAILY_SUMMARY_YYYY-MM-DD.md`
位置：`.commander/`

### 记录内容

**必须包含**:
- 时间（记录日期）
- 当日大事（核心任务、重要成果）
- 关键数据（效率提升、成果统计）
- 时间线（关键事件时间）

**可选**:
- 面临的问题和解决方案
- 下一步计划
- 关键决策理由

### 维护机制

- 每天结束时生成或更新
- 重大事件立即记录
- 使用简洁清晰的格式
- 突出重点（使用emoji、表格）

---

## 🎯 使用方法

### 方式1：手动生成

每天结束时运行：
```bash
# 生成今日总结
python3 .commander/generate_daily_summary.py
```

### 方式2：自动生成（推荐）

设置Cron（每天23:59）：
```cron
59 23 * * * cd ~/.openclaw/workspace/.commander && python3 generate_daily_summary.py
```
