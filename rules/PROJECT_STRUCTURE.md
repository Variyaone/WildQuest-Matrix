# 独立项目文件夹规范

> 所有新建任务必须遵循此规范（USER_COMMANDS.md 2026-03-13）

---

## 📁 标准项目结构

```
project-name/
├── README.md          # 项目说明（必填）
├── docs/              # 文档说明
│   ├── TASK_SPEC.md   # 任务规格
│   └── IMPLEMENTATION.md # 实施方案
├── data/              # 数据文件
├── scripts/           # 脚本文件
├── tests/             # 测试文件
└── reports/           # 报告输出
```

## ✅ 创建检查清单

每个新任务必须：
- [ ] 创建独立项目文件夹 `projects/xxx/`
- [ ] 创建标准目录结构
- [ ] 创建README.md
- [ ] 所有相关文件放入对应目录
- [ ] 禁止文件散落在workspace根目录
